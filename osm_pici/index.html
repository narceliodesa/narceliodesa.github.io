Vias

Wiki oficial de mapeamento de vias

As vias do campus do Pici são classificadas, dentro do modelo do OSM, como vias de serviço.Para mapeá-las usamos a tag:

    highway = service

Podemos combinar a]outras tag’s para adicionar mais características para a via. Como por exemplo:

  1. Adicionar um nome para a via:

    name = optional

  2. Definir o tipo de serviço da via:

    Corredor de estacionamento

    service=parking_aisle

    Ruelas, becos ou vielas sanitárias

    service=alley

    Vias de acesso em caso de emergência

    service=emergency_access

  3. Definir se a via é de mão única ou não:

    oneway=yes

  4. Definir se a pista é coberta:

    covered=yes

  5. Definir o tipo de pavimento da via:

    surface=*

  6. Definir se a via é iluminada:

    lit=yes

  7. Definir as restrições de acesso a via

    access=*
